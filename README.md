## Kata description

See: https://kata-log.rocks/banking-kata

Write a class BankAccount that offers the following methods:
* void deposit(int)
* void withdraw(int)
* String printStatement()

An example statement would be:

|       Date |  Amount | Balance |
|-----------:|--------:|--------:|
| 24.12.2015 | +500.00 |  500.00 |
| 23.08.2016 |  -99.99 |  400.01 |
